Podemos colocar um texto marcado com a tag MARK - agora o texto vai estar marcado.

Pode-se também usar a tag STYLE dentro da tag MARK para alterar a cor do marca texto como agora mesmo eu vou deixa-lo verde.

Podemos diminuir as letras de um texto com a tag SMALL. - Agora o parágrafo vai estar menor

Podemos usar a tag DEL para deixar um texto riscado - como nesse caso

Exemplo de texto sem a tag PRE

Exemplo de texto com a tag PRE

        
            num = int(imput("digite um número"))
            if num % 2 == 0:
                print(f "o número(num) é PAR")
            else:
                print(f "o número(num) é IMPAR")
            print("fim do projeto")
